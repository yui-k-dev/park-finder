# Park Finder

子どもと行ける公園を探すためのWebアプリです。

## 概要

現在地から近くの公園を検索し、地図上で確認できるアプリです。  
また、公園について駐車場の有無や適性年齢などをアンケート形式で回答できる仕組みを想定しています。

ユーザーの回答によって情報を蓄積し、将来的には駐車場の有無や年齢別で検索できるようにすることを目指しています。

---

## 背景

子どもと公園へ行く際に

- 駐車場があるか
- 小さい子どもが遊べるか

といった情報が、既存の地図アプリでは分かりにくいと感じました。  
そのため、ユーザーの回答によって公園情報を集める仕組みを考えました。

---

## 主な機能

- 現在地取得
- 地図表示
- 公園検索
- 公園情報表示
- 駐車場アンケート
- 適性年齢アンケート
